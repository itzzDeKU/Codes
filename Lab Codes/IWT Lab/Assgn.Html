This webpage gives brief introduction to HTML tags.Lets start our introduction to increase familiarity with HTML tags.

  1. Heading
    2. There are several Heading sizes in HTML ranging from H1 to H6 with H1 being largest & H6 being smallest.It is used by enclosing H with its order enclosed in <> like <H1>.

  2. Image
    3. It is used to insert Image in our webpage.It has an attribute "src" to add source of image file.

  3. Subscript & Superscript
    4. It is used to add Subscript and Superscript like H2SO4 & Maths formulae like (a+b)2.

  4. Center
    5. It is used to align the content to Center of the Webpage.

  5. Anchor
    6. It is used as hyperlink and uses its tag href to store URL.Its syntax is <A href ="url">.

  6. Table
    7. It is used to create tabular structure.It has several tags which includes :- thead ,tbody and tfoot. thead is used to group header content.tbody is used to group body content.tfoot is used to group footer content. th is used to define table header of table.tr defines table row and td defines table data. caption is used to give caption to table.

  7. List
    8. It has two types: Unordered list and Ordered list.Ol <ol> is used to create an ordered list.ul <ul> is used to create an unordered list. Both lists use <li> for list item.
